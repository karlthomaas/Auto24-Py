## About Auto24-Py
Auto24-Py is a script, which gathers information from Auto24.ee site (Used/New car market), and 
puts the information into a excel file.

## Used packages:
```
beautifulsoup4==4.9.3
bs4==0.0.1
requests==2.24.0
openpyxl==3.0.7
```
